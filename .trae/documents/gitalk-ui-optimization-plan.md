# Gitalk 评论区 UI 优化计划

## 目标
优化 Gitalk 评论区样式，使其与 Vivia 主题设计风格保持一致，参考主题原有的评论卡片设计。

## 当前问题
1. 评论区没有标题栏和装饰元素
2. 布局与主题其他部分不协调
3. 缺少主题标志性的圆点和标题栏设计

## 优化方案

### 1. 添加评论卡片容器
- 使用 `comment-card` 类包裹 Gitalk 评论区
- 添加标题栏（main-title-bar）和圆点装饰（main-title-dot）
- 标题文字使用 "评论" 或 "Comments"

### 2. 调整评论区宽度
- 与文章正文保持一致的边距
- 使用 `article-side-padding` 变量

### 3. 优化头像样式
- 圆角使用 `card-radius` 或稍大的值
- 头像大小适中（50px 左右）
- 头像位置下移一点，与内容对齐

### 4. 统一字体颜色
- 使用主题的 `--article-text` 和 `--article-meta` 变量
- 确保深浅色模式都能正常显示

### 5. 修复按钮重叠
- 点赞和回复按钮使用 `inline-flex` 布局
- 增加按钮间距

### 6. 参考 Vivia 主题设计
- 参考 Valine/Twikoo 评论区的卡片设计
- 使用相同的标题栏样式
- 保持一致的间距和边距

## 实施步骤
1. 修改 `gitalk.ejs` 添加评论卡片容器
2. 调整 `comment.styl` 样式文件
3. 测试深浅色模式显示效果
4. 验证按钮布局是否正常
